# Sistema-de-efici-ncia-hospitalar
Este projeto realiza a leitura e análise de dados hospitalares a partir de um arquivo CSV, calculando indicadores importantes como taxa de mortalidade e ocupação de leitos. Os resultados são apresentados por meio de visualizações gráficas, facilitando a interpretação da eficiência hospitalar.

## Funcionalidades
- Leitura de dados a partir de arquivo CSV
- Cálculo da taxa de mortalidade por hospital
- Cálculo da taxa de ocupação de leitos
- Gráfico de dispersão relacionando número de leitos e taxa de mortalidade
- Mapa de calor representando a ocupação hospitalar

## Tecnologias Utilizadas
- Python 3
- CSV (biblioteca padrão)
- Matplotlib
- Seaborn

## Objetivo
Projeto desenvolvido com fins educacionais para praticar análise de dados, manipulação de arquivos CSV e visualização gráfica em Python.
